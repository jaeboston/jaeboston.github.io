    How to update the notes

    1. Go to the hugo repository (websitesWS/hugo.notes)
    2. Create/update .md file in the site repository
    3. Or using make.ipynb notebook located in the top level of the hugo repository, convert a notebook to .md file
    4. Add the following content on the top of the file
      ---
  title: "How to update this Note site"
  author: "Jae Song"
  date: 2019-03-20T11:53:49-07:00
  description: "How to update this Note site"
  type: technical_note or page
  draft: false
  ---
    1. Update the index.html in the themes/berbera/layouts directory (usually don’t need to update the index.html file unless you created a new directory in the content folder)
    2. View the site before saving
      • hugo server -D
      • localhost:1313
    3. Check-in the update by commit and push
    4. Publish the web site to the github.io repository
      • hugo -d ../github.io/
    5. Go to the github.io respository
    6. Check-in the updated html codes
    7. View the site
