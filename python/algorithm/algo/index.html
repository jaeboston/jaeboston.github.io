    Tree Terminology

    • height - number of edges(line) between the current node and its furthest leaf

    • level - root is level 1 and root’s children is level 2 etc

    • depth - number of edges to root

    Tree Traverse

    • Depth first search - pre/in/post order apporaches post - childres first

    • [Breath first search] - visiting all nodes at the same level before moving on to child nodes

    BST (binary search tree)

    • left node is always smaller than the right tree
    • unbalanced tree gives you the worst case for the search as O(N)
    • balanced tree gives you the average case for the search as O(lgN)

    Heaps (another type of tree)

    • Min Heaps vs Max Heaps
    • Min Heaps : parent always smaller than children
    • Max Heaps : parent always bigger than children
    • No need to be Binary tree
    • Has to be complete tree meaning all levels excep the last one are completely full
    • Search is lnear time operation O(N) because you don’t know which node to traverse
    • Insert operation is worst case is O(lgN) because insert a node then heapify (heapify takes O(lgN))

    pow(x,y)

    Iteratively solve pow function. Key point: use the property of exponential : for even number 3^4 = (3^2)*(3^2); for odd number 3^5 = (3^4)*2 same as the even number case multiply by x

    def pow(x,y) :
  if y < 0 :
    x = 1/x
    y = -y
  
  result = 1
  while y:
    if y % 2 == 1:
      result * x   #: this line will run once for y = even; twice for y = odd
    
    x *= x
    y = int(y/2)
  return result

    Array (list for Python) algorithm

    • To use space complemity as O(1), don’t try to resize the array (list) to fit the answer. Instead think of using just index number to answer your questions ex) remove the repated number in a sorted list and return the size https://leetcode.com/problems/remove-duplicates-from-sorted-array/submissions/
    def (a: list[int]):
  if len[a] == 0
    return 0
  
  write_index = 1
  for i in range(1, len(a)):
    if a[i] != a[i-1]:
      a[write_index] = a[i]
      write_index += 1
  return write_index
